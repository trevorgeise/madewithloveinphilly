ABOUT THE BADGE

After some extensive searching I couldn't find any type of 'made in philly' badge. That seemed silly so I built one for the launch of salesbox.io.

Now I'd love for anyone else interested to use it as well. Perhaps down the road I'll build out a little site where all Philly startups can be listed.
